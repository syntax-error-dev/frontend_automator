# AI-Powered Frontend Automator

A Python-based automation engine that streamlines content management and site generation using Playwright and Google Gemini AI.

## Features
- **Asynchronous Processing:** Built with `asyncio` for high-performance browser automation.
- **AI-Driven Validation:** Integration with Google Gemini Flash for real-time HTML cleanup and content correction.
- **Modular Design:** Professional project structure with separated concerns (AI, Browser, Utils).
- **Automation:** Handles bulk site generation and content updates via ODS data sources.

## Tech Stack
- Python 3.10+
- Playwright
- Google Generative AI (Gemini)
- Pandas
